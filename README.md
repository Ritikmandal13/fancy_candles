## Getting Started

1. **Install dependencies:**
   ```sh
   npm install
   ```
2. **Start the dev server:**
   ```sh
   npm run dev
   ```
3. **Build for production:**
   ```sh
   npm run build
   ```

## Deployment
Deploy to Vercel, Netlify, or your preferred static host. Set up your Sanity project and environment variables as needed.
